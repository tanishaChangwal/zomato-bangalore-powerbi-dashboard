# Zomato Bangalore Restaurant Analytics Dashboard

# Project Overview
This project presents an interactive Power BI dashboard built using the Zomato Bangalore restaurant dataset. The goal of this project is to analyze restaurant trends, customer preferences, pricing patterns, and location-wise performance across Bangalore.

The dashboard helps uncover key insights related to:
- Restaurant distribution by location
- Customer ratings and votes
- Popular cuisines
- Online ordering availability
- Table booking availability
- Cost vs Rating relationship

# Tools Used
- Power BI
- Power Query
- DAX
- Excel / CSV Dataset

# Dataset Information
The dataset contains restaurant information from Bangalore, including:
- Restaurant Name
- Location
- Rating
- Votes
- Cuisines
- Approx Cost
- Online Order Availability
- Table Booking Availability
- Restaurant Type

# Data Cleaning & Transformation
Data cleaning was performed in Power Query:
- Removed duplicates
- Handled null/missing values
- Cleaned rating column
- Converted cost column into numeric format
- Created cost ranges for segmentation
- Improved column formatting

# Dashboard Pages
# 1. Executive Overview
Provides high-level business metrics:
- Average Rating
- Average Cost
- Total Restaurants
- Total Votes
- Restaurant distribution by location
- Online order and table booking availability

# 2. Customer Preference & Performance
Focused on customer behavior and restaurant performance:
- Top restaurants by votes
- Restaurant rating vs votes analysis
- Ratings by restaurant type
- Location-wise performance

# 3. Cost & Cuisine Insights
Analyzes pricing and cuisine trends:
- Most popular cuisines
- Average rating by location
- Average cost by location
- Cost vs rating analysis

# Key Insights
- Whitefield has the highest restaurant concentration.
- Online ordering is available in most restaurants.
- Average restaurant rating across Bangalore is approximately 3.7.
- Higher-rated restaurants generally receive higher customer engagement.
- Cost and ratings show a moderate positive relationship in premium restaurants.

# Business Impact
This dashboard helps businesses and analysts:
- Identify high-performing restaurant locations
- Understand customer preferences
- Analyze pricing strategies
- Improve decision-making using data-driven insights
